# TypeScript BoilerPlate

## Description

this is a boilerplate for typescript projects

- SWC
- ESlint
- Prettier
- Jest

## Installation

```bash
$ git clone
$ yarn install
```

## Usage

- customize package.json scripts for your project

### Start

```bash
$ yarn start
```

### Build

```bash
$ yarn build
```

### Test

```bash
$ yarn test
```
